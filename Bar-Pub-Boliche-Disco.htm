Capturas del sistema

¿Estas buscando un software para bar? no busques más, llegaste al lugar adecuado para obtener tu software de bar. El sistema Núcleo IT Gastronómico es un software que gestiona eficientemente los pedidos del bar o boliche. Con el programa se logra un estricto control de empleados (mozos, cajeros y encargados) y se optimiza la toma de pedidos para lograr una mejor atención para con los clientes.

El control de stock con alarmas (cantidades mínimas de stock) y el manejo de recetas permiten planificar las compras de ingredientes para mantener el negocio operativo al 100%. El control de stock es muy detallado y es ideal para controlar al personal ya que cuenta con un resumen de movimientos de stock similar al resumen bancario (justifica el stock actual de cada producto e ingrediente). En cualquier momento, de una forma rápida y sencilla se puede realizar un Inventario de stock y así detectar posibles desviaciones de productos o materias primas.

Emite comandas para la preparación de cada plato, puede imprimirse en una comandera ubicada en la cocina y otra en la barra, para la parte de cafetería o bebidas, comprobantes de venta no fiscales (pre-tickets) y tickets fiscales.

Todas estas funcionalidades las puede realizar una persona sin conocimientos de informática ya que el sistema es muy simple e intuitivo.

Contamos con personal especializado para asistirlo antes y después de haber implementado el software Núcleo IT Gastronómico, dando así un servicio Premium tanto en la pre venta como en la post venta.


Integraciones

Buscando generar mayores beneficios a nuestros usuarios, integramos nuestro sistema Núcleo IT Gastronómico con las siguientes empresas

  • Facturación

    Según su necesidad podrá facturar a través de una tickeadora fiscal o electrónicamente a través del Web Service de AFIP cumpliendo con la Resolución General Nº3749.

  • Ventas y Compras

    La carga de las ventas y compras descuentan y aumentan el stock de los productos, los saldos de la caja y permiten gestionar las cuentas correientes de los clientes y proveedores.

  • Control de Stock

    Con el sistema podrá controlar el stock de su negocio, tanto de los productos como de los ingredientes que los componen. El programa cuenta con alarmas de stock que lo ayudarán a saber qué reponer y cuanto de cada uno.

  • Control de Caja

    Controlar la caja del negocio con el sistema le permitirá saber en cualquier momento con cuanta plata cuenta. Podrá configurar un cierre ciego para que los empleados no sepan cuanta plata debería haber.

  • Estadísticas

    Viendo todos los reportes estadísticos con los que cuenta el software podrá analizar y gestionar de mejor manera su negocio.

  • Control Online

    Con este servicio ud. podrá ver las principales estadísticas del software desde cualquier lugar (PC, tablet o celular) a través de internet. Esto también le servirá para tener un backup automático prácticamente en tiempo real.

  • Fácil de usar

    El programa fue pensado y diseñado para que cualquier persona, pueda manejarlo sin problema. Lo podrá hacer solo con teclado, con mouse o con pantalla táctil.

Preguntas Frecuentes

¿Se puede utilizar con pantalla táctil?

Sí, el sistema fue diseñado para poder ser utilizado eficientemente durante la toma de pedidos y la atención en el salón con pantalla táctil.

¿Se puede usar con teclado?

Todos los sistemas que desarrollamos permiten una forma de manejo rápida y dinámica. Es posible realizar la toma de pedidos de forma completa con el teclado, sin la necesidad de utilizar el mouse.

¿Se puede utilizar con varias comanderas?

Sí, se pueden utilizar tantas comanderas como sea necesario. Inclusive el sistema permite especificar a que comandera van a ir las comandas de cada rubro. Por ejemplo, permite especificar que los productos que pertenecen al rubro de las bebidas se impriman en la comandera de la barra.

¿Se puede realizar el control del negocio online?

El sistema permite controlar el estado del negocio, accediendo a un sitio web con su usuario y contraseña. Dentro del sitio se podrán visualizar los principales datos del negocio para realizar un análisis completo del local.

¿Se puede facturar desde el sistema?

Sí, se puede facturar a través de tickeadoras fiscales o electrónicamente a través del Web Service de AFIP cumpliendo con la Resolución General Nº3749.

¿Se puede utilizar en red? ¿Lo comercializan por PC?

El software funciona en red dado que el mismo es multiusuario. Con la compra del programa se entrega 1 (una) licencia de uso para poder instalarlo hasta en 1 (una) PC. En caso de necesitar instalarlo en más de una computadora las licencias adicionales pueden ser adquiridas en cualquier momento.

¿Cuento con soporte?

Sí, Ud. puede comunicarse con la mesa de ayuda por teléfono, e-mail o Skype. Y ahora también sumamos al WhatsApp como medio de comunicación.
